# 战疫小游戏

## 项目简介

我将从零开发一个 H5 游戏，主要使用 phaser3 来制作的游戏。结合当下疫情的严峻形式，我也将一些元素融入到这款游戏中，同时希望疫情早日结束，早点摘下口罩，可以看到彼此脸上洋溢的笑容。

元素一：出门要戴口罩
元素二：为生活打拼，是收集粮食
元素三：奋勇拼搏，要打死恶魔怪物，与各种黑势力做斗争
