# 坚持到底
## 项目简介
该项目是使用 Flutter 基于 Flame 游戏引擎实现的一个坚持到底的小游戏。
### 游戏规则
- 点击游戏中间的 START 开始游戏；
- 游戏开始后四周会出现大小不一的子弹并以一定速度向目标移动，且随着游戏时长的增加，子弹的移动速度越快，玩家可拖拽目标躲避子弹，当子弹与目标产生碰撞时游戏结束；
- 游戏的目的就是挑战坚持的游戏时长，游戏时长越长成绩越高。
