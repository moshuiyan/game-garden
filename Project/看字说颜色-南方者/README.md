#### 游戏规则
① 看图模式，根据窗体生成的字与字体颜色，说出对应的字体颜色。（字是干扰项） 
② 答题模式，根据题目要求；选出正确答案。

#### 干扰等级
入门（5种颜色）、初级（8种颜色）、中级（10种颜色）、高级（12种颜色）.
