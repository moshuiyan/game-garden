# 项目简介

## 扫雷游戏介绍

**游戏规则：**
- 在不触碰到任何地雷的情况下发现一个雷区。

**游戏玩法：**
- 揭开一个没有地雷的快之后，将会显示一个数字来表示周围地雷的数量，

- 在不触碰任何地雷的情况下揭开所有格子即为胜利。
