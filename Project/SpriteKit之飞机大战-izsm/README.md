# 项目简介
一款iOS端使用的飞机大战游戏，该游戏使用`SpriteKit`2D引擎实现的
#### 玩法
>1、该游戏为闯关游戏，玩家拖动飞机消灭敌人来获取分数<br>
>2、初始的时候玩家有3条生命，每一关会随机掉落一个血包和增加飞机的攻击力。敌人的生命值也是随着闯关级数逐级增加的，每一关都会有一个boss，打败boss会获得大量分数
